## 2. The API (delivered, kernel-validated — `Agora/SymSquare.lean`)

```
structure DiffOp2 where p q : Polynomial ℚ        -- monic  D² + p·D + q
structure DiffOp3 where b c d : Polynomial ℚ      -- monic  D³ + b·D² + c·D + d
noncomputable def symSquare (L : DiffOp2) : DiffOp3
def IsSymSquareOf (L3 : DiffOp3) (L2 : DiffOp2) : Prop := L3 = symSquare L2
```

`symSquare` uses the classical formula
`Sym²(D²+p·D+q) = D³ + 3p·D² + (2p²+p'+4q)·D + (4pq+2q')`,
**kernel-validated** by `symSquare_golden_harmonic` and `symSquare_golden_exp` (two
first-principles solution-space examples, not memory). If you change the formula, those
golden tests are the gate.

**Why C3 is non-vacuous here (contrast E-002):** `IsSymSquareOf` pins *every* coefficient
of L3 as an explicit polynomial in L2's coefficients. It is a specific, per-candidate
falsifiable identity — not `∃ P, P.natDegree = 3` (true of any cubic).
